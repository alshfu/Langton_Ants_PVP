# Grafana dashboards

Положить сюда JSON-файлы дашбордов для:
- API overview (RPS, p50/p95/p99, error rate)
- WebSocket overview (active connections, messages/sec, backpressure drops)
- Game workers (active matches, tick duration, memory)
- Matchmaking (queue sizes by region/mode, ETA, match found rate)
- Business KPI (DAU, MAU, retention)

Источник вдохновения: grafana.com/dashboards/ + node_exporter, postgres_exporter дашборды.
